Evebury.Gdsn.Gs1

Featured
.NET 10 MIT License GDSN v3.1.33

A comprehensive .NET library for validating GDSN GS1 messages. Acts as a pre-validation layer that catches data quality issues before your messages reach the GS1 network — saving you API costs, rejection headaches, and hours of manual troubleshooting.

Pre-validation Engine

Catch errors before they reach GS1, reducing API costs and rejected messages.

Multi-language Messages

Human-readable validation errors in English, Dutch, French, and German.

Smart Delta Updates

Efficiently compare and sync only changed data across your catalog.

Zero Dependencies

Pure .NET implementation — no third-party packages, no surprises.

// Install via NuGet Package Manager
dotnet add package Evebury.Gdsn.Gs1
using Evebury.Gdsn.Gs1;

// Validate a GDSN message before sending to GS1
var validator = new GdsnValidator();
var result = validator.Validate(gdsnMessage);

if (!result.IsValid)
{
    foreach (var error in result.Errors)
    {
        Console.WriteLine(error.Message);
        // "GTIN check digit is invalid for '8710123456780'"
    }
}

Evebury.Gs1.DigitalLink

Featured
.NET 10 MIT License GS1 Digital Link

A .NET library for generating strongly typed and validated GS1 Digital Link URIs. Build standards-compliant digital links with a fluent API — complete with check digit validation, custom domains, and built-in support for trade items, dates, weights, and prices.

Strongly Typed Builder

Type-safe builder pattern with compile-time checks for all GS1 Application Identifiers.

Automatic Validation

Check digit validation and clear error messages for invalid inputs.

Rectified Segments

Simplified subset of Application Identifiers for common use cases.

Custom Domains

Use your own branded domain for GS1 Digital Link URIs.

// Install via NuGet Package Manager
dotnet add package Evebury.Gs1.DigitalLink
using Evebury.Gs1.DigitalLink;

// Build a GS1 Digital Link URI with fluent API
var link = DigitalLinkBuilder
    .Create("https://your-domain.com")
    .WithGtin("08710123456789")
    .WithBatchLot("ABC123")
    .Build();

Console.WriteLine(link.Uri);
// https://your-domain.com/01/08710123456789/10/ABC123

Frequently Asked Questions

Evebury.Gdsn.Gs1 is a .NET library that validates your GDSN messages before they reach the GS1 network. Think of it as a spell checker for product data — it catches invalid GTINs, missing required fields, and format errors locally, so you avoid costly API rejections and hours of manual troubleshooting. It supports GDSN v3.1.33 and provides validation messages in four languages.
Evebury.Gs1.DigitalLink helps you generate standards-compliant GS1 Digital Link URIs using a strongly typed, fluent API. It handles check digit validation, supports all common Application Identifiers (GTIN, batch/lot, serial numbers, dates, weights, prices), and lets you use your own branded domain for the URIs.
